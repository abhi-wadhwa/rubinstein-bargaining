# Rubinstein Bargaining

Implementation of the Rubinstein alternating-offers model with finite-horizon, outside options, and Nash bargaining convergence.

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/rubinstein-bargaining.git
cd rubinstein-bargaining
make install
make run
make test
```

See [docs/](../docs/rubinstein-bargaining.tex) for the math.
